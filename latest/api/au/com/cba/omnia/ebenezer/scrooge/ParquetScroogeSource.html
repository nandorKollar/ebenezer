ParquetScroogeSource

Related Docs: object ParquetScroogeSource | package scrooge

class ParquetScroogeSource[T <: ThriftStruct] extends FixedPathSource with TypedSink[T] with Mappable[T] with Serializable

A scalding source to read Scrooge Thrift structs from partitioned files using parquet as the underlying storage format. It will ignore the partition columns and only read the thrift struct from the parquet file.

Use PartitionHiveParquetScroogeSink for write.
